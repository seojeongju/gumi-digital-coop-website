# 🚀 새 세션 빠른 시작 가이드

## 📍 현재 위치
**후이즈 "네임서버 고급설정" 화면**에서 작업 중단됨

---

## ⚡ 즉시 해야 할 3가지 작업

### 1️⃣ DNS 호스트 관리 접속
```
현재 화면에서 "DNS 호스트 설정" 버튼 클릭
```

### 2️⃣ gdamic.kr 루트 CNAME 삭제
```
삭제: gdamic.kr → gumi-digital-coop-website.pages.dev
유지: www.gdamic.kr → gumi-digital-coop-website.pages.dev
```

### 3️⃣ URL 포워딩 설정
```
gdamic.kr → https://www.gdamic.kr (301 리다이렉트)
```
